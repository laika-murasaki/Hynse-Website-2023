Creative เซิร์ฟเวอร์ / Totem Void 

Feature ใหม่ใน Patch 3

- เซิร์ฟเวอร์ Creative
- เซิร์ฟเวอร์ Game
- จุดมาร์ด้วย Banner ในเกม
- ใช้ Totem ใน Void
- แผนที่ทุกเซิร์ฟเวอร์
- อัปโหลด schematica เข้าเซิร์ฟเวอร์ Creative ผ่านเว็บ
- Anti Villager Lag
- Config Server for nerd
- สถานะเซิร์ฟเวอร์ใน Debug Menu หรือ F3
- ไอเท็ม Frame หล่องหน

การเปลี่ยนแปลงใน Patch 3

- หน้าเว็บ ส่วนเสริม และ อัพเตด แนะนำให้เข้าไปดู
- แผนที่ทุกเซิร์ฟเวอร์
- Reputation แก้การปั้มให้ได้แค่ 1 คน ต่อ 1 ครั้ง และ ใส่เหตุผลพิเศษได้
- ลบระบบนั่งออก
- แชทรวมกันทุกเซิร์ฟเวอร์สามารถพูดคุยข้ามเซิร์ฟเวอร์ได้และส่วนตัวก็ได้
- แทบรายชื่อผู้เล่นจะรวมกันทุกเซิร์ฟเวอร์
